You might have heard of self hosted GitLab. Well, I set it up!
If you visit [https://git.anirudhsevugan.me/](https://git.anirudhsevugan.me/), you will be redirected to an Ngrok app that is forwarding to a localhost being run
via Docker on my computer.

The server will rarely be up though, so if you want it to be you can contact me at [personal@anirudhsevugan.me](mailto:personal@anirudhsevugan.me)
